This download link will download a ZIP file containing a Python file (flag-demo.py) which, when run, will allow you to enter text and see if it gets flagged. You must unzip the directory somewhere known.

Please note, Python 3 is required for this demo. The demo was written in Python 3.9.5. In order to initialize the program (and install any dependencies) you must run the following terminal command: pip install -r requirements.txt.
