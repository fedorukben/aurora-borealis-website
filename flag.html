Flag Phase Demonstration

This download link will download a ZIP file containing a Python file (flag-demo.py) which, when run, will allow you to enter text and see if it gets flagged. You must unzip the directory somewhere known.

Please note, Python 3 is required for this demo. The demo was written in Python 3.9.5. In order to initialize the program (and install any dependencies) you must run the following terminal command:

pip install -r requirements.txt

You must then use the following command to run the file:

python3 flag-demo.py

Please keep in mind that you must ensure that your present working directory is the 'flag-demo' folder.

Please feel free to watch the following installation guides on YouTube to install the software. We have created installation guides for Windows, macOS, and Ubuntu/Linux.

If you are having troubles installing the program or do not wish to install the software, we invite you to click here to watch a YouTube video demonstration of the program.
